class torch.nn. Softmax2d [source] ¶

Applies SoftMax over features to each spatial location.

When given an image of Channels x Height x Width , it will apply Softmax to each location ( C h a n n e l s , h i , w j ) (Channels, h_i, w_j) ( C hann e l s , h i ​ , w j ​ )

Shape:

  • Input: ( N , C , H , W ) (N, C, H, W) ( N , C , H , W ) or ( C , H , W ) (C, H, W) ( C , H , W ) .

  • Output: ( N , C , H , W ) (N, C, H, W) ( N , C , H , W ) or ( C , H , W ) (C, H, W) ( C , H , W ) (same shape as input)

Returns

a Tensor of the same dimension and shape as the input with values in the range [0, 1]
