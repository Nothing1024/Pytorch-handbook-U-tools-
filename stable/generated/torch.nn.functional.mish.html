torch.nn.functional.mish ¶

torch.nn.functional. mish ( input , inplace = False ) [source] ¶

Applies the Mish function, element-wise. Mish: A Self Regularized Non-Monotonic Neural Activation Function.

Mish ( x ) = x ∗ Tanh ( Softplus ( x ) ) \text{Mish}(x) = x * \text{Tanh}(\text{Softplus}(x)) Mish ( x ) = x ∗ Tanh ( Softplus ( x ))
