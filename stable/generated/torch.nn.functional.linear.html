torch.nn.functional.linear ¶

torch.nn.functional. linear ( input , weight , bias = None ) [source] ¶

Applies a linear transformation to the incoming data: y = x A T + b y = xA^T + b y = x A T + b .

This operator supports TensorFloat32 .
